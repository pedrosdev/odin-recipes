
# Introduction
Single-page website containing links to a few recipes.

## Learning goals:
Get familiar with basic HTML elements such as the basic boilerplate
and things like lists (ol and ul), headings, paragraphs, etc.

## Skills:
 - Basic git workflow
 - HTML fundamentals
